What is XML and JavaScript?

What is XML?

XML which stands for Extensible Markup Language is a markup language that is used for storing and transporting data.
It is very similar to HTML.

How is XML Used?

XML can be used in the following instances:

  1. Configuration Files
  2. Document Formats
  3. Data Exchange
  4. Data Storage
  5. Web Services
  6. Metadata

What is JavaScript?

JavaScript is an object based, client side scripting language interpreted by a web browser
It is object-based because it is used to work with the objects associated with a web page document.
It is considered a client side scripting language because it is interpreted by a browser.

How is JavaScript related to HTML and CSS?

JavaScript is related to HTML and CSS because

  • JavaScript makes the content interactive
  • HTML defines the structure of your content
  • CSS determines the style and layout

What are some common JavaScript elements, properties, methods, and functions?

JavaScript Elements

Common JavaScript Elements include:

  • Variables and Data Types
  • Control Structures
  • Error Handling
  • Operators

JavaScript Properties

A JavaScript property is a member of an object that associates a key with a value
JavaScript Property Examples: String Property, Array Property, etc.

JavaScript Methods

JavaScript methods are actions that can be performed on objects
JavaScript Method Examples: toUpperCase(), charAt(), indexOf()

JavaScript Functions

A JavaScript function is a block of code designed to perform a particular task.
